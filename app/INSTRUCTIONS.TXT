when creating a new js file follow this:

- add the file to your index.html
- add the dependency to the module in app.js


=== Forms
- need ng-model on the input to validate!
- the ng-model name must match the name of the $scope to bind (See add keyword or add city inputs and list scope )


=== Making a JSON call from to a local file

 $http.get('partials/tweets/cupcakes.json')
    .success(function(data){
   		$scope.cupcakes = data;
    })
    .error(function(data){
       //error
    })
